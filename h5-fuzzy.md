# X.2) Fuff project [README](<https://github.com/ffuf/ffuf/blob/master/README.md>)
- A fast web fuzzer written in Go
- You can use ffuf for GET and POST requests, you just specify what to FUZZ <-- this works again as the keyword!
- Use the keyword anywhere in the URL `-u`, headers `-H` or POST data `-d`
- Press `enter` during execution to enter interactive mode to reconfigure your filters etc.
